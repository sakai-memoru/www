## *** overview
- GitHub Pagesで、markdownファイルを、Jekyll Themeで公開する。

## *** procedure

### ** note
- GitHubにrepositoryを作り、そのrepositoryを、GitHub Pagesとして公開。
- GitHub RepositoryのREADME.mdを修正して、公開。
